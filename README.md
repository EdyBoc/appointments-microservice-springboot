Citas Service – Microservicio Spring Boot

Microservicio desarrollado con Java + Spring Boot, encargado de administrar el módulo de citas médicas mediante un API REST limpio, modular y fácil de escalar.
Incluye validaciones, DTOs, mapeos, servicios desacoplados y acceso a base de datos con JPA/Hibernate.

Tecnologías utilizadas:

-Tecnología	Uso
-Java 8+	Lenguaje principal
-Spring Boot	Framework principal
-Spring Web	Controladores REST
-Spring Data JPA	Acceso a datos
-Hibernate	ORM
-MySQL / PostgreSQL	Base de datos
-Validation API	Validaciones con anotaciones
-Maven	Gestión de dependencias
